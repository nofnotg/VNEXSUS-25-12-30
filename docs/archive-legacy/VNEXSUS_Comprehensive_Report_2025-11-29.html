Key Components
  • Frontend: Vanilla JS + Bootstrap 5. Lightweight, responsive UI for file uploads and report viewing.
  • Backend: Node.js + Express. Handles API requests, file processing, and orchestration of AI services.
  • DNA Engine: Custom logic for "sequencing" medical text into structured data (dates, diagnoses, treatments).
  • AI Layer: Hybrid approach using GPT-4 for general reasoning and Claude 3.5 Sonnet for deep context analysis.
  • Automation: Make.com integration for generating polished Google Docs/PDF reports.
