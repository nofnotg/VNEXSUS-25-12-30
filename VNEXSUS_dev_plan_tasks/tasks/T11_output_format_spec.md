# T11. UI 출력 규격 반영(Core/Episode/Full + 질문맵)

## 목적
- VNEXSUS txt가 “심사기준 우선”으로 읽히도록 섹션 순서 및 표현을 표준화.

## 출력 섹션(권장)
1) 케이스 메타(가입일/조회기간/주요 청구 병명(가능 시))
2) 가입 전 3개월 핵심 이벤트(Core)
3) 가입 전 5년 핵심 이벤트(Core)
4) 고지의무 질문 맵(Question Map)
5) Episode Summary(기간/횟수/대표 이벤트)
6) Full Timeline(전체 이벤트, 날짜순)

## 완료 기준(DoD)
- 케이스 간 출력 포맷이 일관됨(조사자 복사/붙여넣기 효율)
- 모든 Core 이벤트에 근거 스팬 포함
