┌─────────────────────────────────────────────────────────┐
│  🚀 가장 쉬운 방법 (Docker 사용 - 추천!)                  │
└─────────────────────────────────────────────────────────┘

1️⃣ Docker Desktop 설치
   👉 https://www.docker.com/products/docker-desktop/
   - "Download for Windows" 클릭하여 다운로드
   - 설치 후 Docker Desktop 실행

2️⃣ API 키 설정
   - ".env.secure" 파일을 복사하여 ".env"로 이름 변경
   - ".env" 파일을 메모장으로 열기
   - "GOOGLE_CLOUD_VISION_API_KEY" 항목에 API 키 입력
   - 파일 저장

3️⃣ 앱 실행
   - "start-docker.bat" 파일을 더블클릭
   - 잠시 기다리면 자동으로 실행됩니다

4️⃣ 브라우저에서 접속
   - Chrome, Edge, Firefox 등 브라우저 열기
   - 주소창에 입력: http://localhost:3030

5️⃣ 앱 종료
   - "stop-docker.bat" 파일을 더블클릭


┌─────────────────────────────────────────────────────────┐
│  💻 Node.js로 직접 실행하기                               │
└─────────────────────────────────────────────────────────┘

1️⃣ Node.js 설치
   👉 https://nodejs.org/
   - "LTS" 버전 다운로드 및 설치

2️⃣ 자동 설정 실행
   - "windows-setup.bat" 파일을 우클릭
   - "관리자 권한으로 실행" 선택
   - 화면의 안내를 따라 진행

3️⃣ API 키 설정
   - ".env" 파일을 메모장으로 열기
   - "GOOGLE_CLOUD_VISION_API_KEY" 항목에 API 키 입력
   - 파일 저장

4️⃣ 서버 실행
   - "start-server.bat" 파일을 더블클릭

5️⃣ 브라우저에서 접속
   - 주소창에 입력: http://localhost:3030

6️⃣ 서버 종료
   - 실행 중인 콘솔 창에서 Ctrl+C 누르기


┌─────────────────────────────────────────────────────────┐
│  📖 접속 주소                                            │
└─────────────────────────────────────────────────────────┘

메인 애플리케이션:   http://localhost:3030
파일 업로드:         http://localhost:3030/hybrid-interface.html
개발 관리자:         http://localhost:8088


┌─────────────────────────────────────────────────────────┐
│  🔧 다른 컴퓨터에서 접속하기                              │
└─────────────────────────────────────────────────────────┘

같은 WiFi에 연결된 다른 컴퓨터나 스마트폰에서도 접속 가능!

1️⃣ 내 컴퓨터의 IP 주소 확인
   - 윈도우 검색에서 "cmd" 입력
   - 명령 프롬프트에서 "ipconfig" 입력
   - "IPv4 주소" 확인 (예: 192.168.0.100)

2️⃣ 다른 기기에서 접속
   - 브라우저 주소창에 입력: http://192.168.0.100:3030
   (자신의 IP 주소로 변경)
