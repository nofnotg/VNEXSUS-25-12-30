# VNEXSUS AI 의료 문서 처리 시스템 사용자 가이드

## 🏥 시스템 소개

VNEXSUS AI는 의료 문서의 OCR 처리와 AI 기반 텍스트 보강을 통해 의료진의 업무 효율성을 높이는 지능형 문서 처리 시스템입니다.

### 주요 기능
- **고정밀 OCR 처리**: 의료 문서 전용 최적화된 텍스트 인식
- **AI 프롬프트 보강**: 27.4% 평균 개선율의 텍스트 품질 향상
- **병원별 맞춤 처리**: 각 병원의 문서 형식에 특화된 처리
- **실시간 처리**: 평균 30초 이내 빠른 문서 처리
- **다양한 출력 형식**: JSON, Excel, 텍스트 등 다양한 형식 지원

## 🚀 시작하기

### 1. 시스템 접속

웹 브라우저에서 다음 주소로 접속하세요:
```
http://localhost:3000
```

### 2. 첫 화면 구성

메인 화면에서 다음 요소들을 확인할 수 있습니다:

- **파일 업로드 영역**: 드래그 앤 드롭 또는 클릭하여 파일 선택
- **병원 선택**: 문서가 발행된 병원 선택
- **처리 옵션**: AI 보강, 언어 설정 등
- **진행 상황 표시**: 실시간 처리 진행률 확인

## 📄 문서 업로드 및 처리

### 1. 지원 파일 형식

| 형식 | 확장자 | 최대 크기 | 권장사항 |
|------|--------|-----------|----------|
| PDF | .pdf | 50MB | 텍스트가 포함된 PDF 권장 |
| 이미지 | .jpg, .jpeg, .png | 20MB | 300DPI 이상 권장 |
| TIFF | .tif, .tiff | 30MB | 의료 스캔 문서에 최적 |

### 2. 파일 업로드 방법

#### 방법 1: 드래그 앤 드롭
1. 파일을 파일 탐색기에서 선택
2. 업로드 영역으로 드래그하여 놓기
3. 파일이 자동으로 업로드됨

#### 방법 2: 파일 선택
1. "파일 선택" 버튼 클릭
2. 파일 탐색기에서 원하는 파일 선택
3. "열기" 버튼 클릭

### 3. 처리 옵션 설정

#### 병원 선택
- **강북삼성병원**: 강북삼성병원 문서 형식에 최적화
- **은평성모병원**: 은평성모병원 문서 형식에 최적화
- **서울병원**: 서울병원 문서 형식에 최적화
- **일반 클리닉**: 일반적인 의료 문서 형식

#### AI 보강 옵션
- **기본 보강**: 기본적인 텍스트 정리 및 오타 수정
- **고급 보강**: 의료 용어 정규화 및 구조화
- **프리미엄 보강**: 최고 품질의 텍스트 개선 (처리 시간 증가)

#### 언어 설정
- **한국어**: 한국어 의료 문서 (기본값)
- **영어**: 영어 의료 문서
- **자동 감지**: 언어 자동 감지

### 4. 처리 시작

1. 파일 업로드 완료 확인
2. 병원 및 옵션 선택
3. "처리 시작" 버튼 클릭
4. 진행률 바에서 처리 상황 확인

## 📊 처리 결과 확인

### 1. 결과 화면 구성

처리 완료 후 다음 정보를 확인할 수 있습니다:

#### 처리 요약
- **처리 시간**: 총 소요 시간
- **페이지 수**: 처리된 페이지 수
- **신뢰도**: OCR 처리 신뢰도 (0-100%)
- **개선율**: AI 보강으로 인한 텍스트 개선 정도

#### 텍스트 결과
- **원본 텍스트**: OCR로 추출된 원본 텍스트
- **보강된 텍스트**: AI로 개선된 최종 텍스트
- **변경 사항**: 원본과 보강된 텍스트의 차이점 하이라이트

### 2. 결과 다운로드

#### 다운로드 형식 선택
- **JSON**: 구조화된 데이터 형식 (개발자용)
- **Excel**: 표 형식으로 정리된 데이터
- **텍스트**: 순수 텍스트 파일
- **PDF**: 원본과 결과를 비교할 수 있는 PDF

#### 다운로드 방법
1. 원하는 형식 선택
2. "다운로드" 버튼 클릭
3. 브라우저 다운로드 폴더에 파일 저장

### 3. 결과 공유

#### 링크 공유
- "공유 링크 생성" 버튼 클릭
- 생성된 링크를 복사하여 공유
- 링크는 24시간 동안 유효

#### 이메일 전송
- "이메일로 전송" 버튼 클릭
- 수신자 이메일 주소 입력
- 결과가 첨부된 이메일 자동 발송

## 🔧 고급 기능

### 1. 배치 처리

여러 파일을 한 번에 처리할 수 있습니다:

1. "배치 처리" 탭 선택
2. 여러 파일을 드래그 앤 드롭으로 추가
3. 공통 설정 적용
4. "일괄 처리 시작" 버튼 클릭

### 2. 템플릿 관리

자주 사용하는 설정을 템플릿으로 저장:

1. "템플릿 관리" 메뉴 선택
2. "새 템플릿 만들기" 클릭
3. 템플릿 이름 및 설정 입력
4. "저장" 버튼 클릭

### 3. 처리 히스토리

과거 처리 내역을 확인하고 재다운로드:

1. "히스토리" 메뉴 선택
2. 날짜별 또는 파일명으로 검색
3. 원하는 항목 선택하여 결과 재확인

## 📈 품질 개선 팁

### 1. 최적의 파일 준비

#### PDF 파일
- **텍스트 PDF 권장**: 스캔된 이미지보다 텍스트가 포함된 PDF가 더 정확
- **해상도**: 최소 150DPI, 권장 300DPI 이상
- **파일 크기**: 50MB 이하로 유지

#### 이미지 파일
- **해상도**: 300DPI 이상 권장
- **형식**: PNG > JPEG > GIF 순으로 권장
- **색상**: 흑백 또는 그레이스케일이 더 정확
- **조명**: 균일한 조명, 그림자 없이 촬영

### 2. 병원 선택의 중요성

올바른 병원을 선택하면 처리 정확도가 크게 향상됩니다:

- **강북삼성병원**: 41개 전용 패턴으로 99.92% 노이즈 제거
- **은평성모병원**: 20개 전용 패턴으로 99.82% 노이즈 제거
- **일반 클리닉**: 21개 범용 패턴으로 99.77% 노이즈 제거

### 3. AI 보강 레벨 선택

| 레벨 | 처리 시간 | 개선 효과 | 권장 용도 |
|------|-----------|-----------|----------|
| 기본 | 빠름 | 기본적 오타 수정 | 간단한 문서 |
| 고급 | 보통 | 의료 용어 정규화 | 일반적인 의료 문서 |
| 프리미엄 | 느림 | 최고 품질 개선 | 중요한 문서 |

## 🚨 문제 해결

### 1. 일반적인 문제

#### 파일 업로드 실패
**증상**: 파일이 업로드되지 않음
**해결방법**:
- 파일 크기 확인 (50MB 이하)
- 지원 형식 확인 (PDF, JPG, PNG, TIFF)
- 브라우저 새로고침 후 재시도

#### 처리 시간 지연
**증상**: 처리가 오래 걸림
**해결방법**:
- 파일 크기 줄이기 (이미지 압축)
- AI 보강 레벨 낮추기
- 네트워크 연결 상태 확인

#### 결과 품질 불만족
**증상**: OCR 결과가 부정확함
**해결방법**:
- 원본 파일 품질 개선
- 올바른 병원 선택
- AI 보강 레벨 높이기

### 2. 오류 메시지 해결

#### "파일 형식이 지원되지 않습니다"
- 지원되는 형식: PDF, JPG, JPEG, PNG, TIFF
- 파일 확장자 확인
- 파일이 손상되지 않았는지 확인

#### "처리 중 오류가 발생했습니다"
- 파일 크기 줄이기
- 다른 파일로 테스트
- 관리자에게 문의

#### "서버에 연결할 수 없습니다"
- 인터넷 연결 확인
- 방화벽 설정 확인
- 잠시 후 재시도

### 3. 성능 최적화

#### 브라우저 설정
- **권장 브라우저**: Chrome, Firefox, Edge
- **JavaScript 활성화**: 필수
- **쿠키 허용**: 세션 유지를 위해 필요

#### 네트워크 환경
- **안정적인 연결**: WiFi 또는 유선 인터넷
- **충분한 대역폭**: 업로드/다운로드를 위해 필요
- **방화벽 예외**: 포트 3000, 3030 허용
