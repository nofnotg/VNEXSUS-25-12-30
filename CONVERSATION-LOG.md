## 🔍 **핵심 문제들과 해결책**

### **Problem 1: 레이아웃 vs 선형 텍스트**
**문제 설명**:
```
원본 표 형태:
┌─────────────┬──────────────┐
│ 2022-03-15  │ 서울대병원   │
│ 진단        │ 급성충수염   │
└─────────────┴──────────────┘

OCR 추출:
"2022-03-15 서울대병원 진단 급성충수염"
```

**해결책**: 컨텍스트 윈도우 분석
- 전후 200자 문맥으로 구조 추론
- AI가 원본 레이아웃 복원
- 정보의 계층 구조 파악

### **Problem 2: 다양한 병원 양식**
**패턴 A**: `날짜 → 데이터`
**패턴 B**: `데이터 → 날짜 → 데이터`

**해결책**: 적응형 패턴 학습
- 문서별 레이아웃 패턴 분석
- 날짜와 데이터의 소속 관계 파악
- 진화형 학습으로 새 양식 적응

### **Problem 3: 중첩된 날짜**
**문제**:
```
주 사건: 2025-05-10 진료
내부 언급: "2025-04-30 치료 받고 가셨음"
```

**해결책**: 주/부 날짜 계층 분석
- 섹션을 나누는 주요 날짜 vs 언급된 날짜
- 현재 기록 vs 과거 사건 참조 구분
- 정보 소속 관계 명확화

### **Problem 4: 인과관계 하이퍼링크**
**해결책**: 의료 사건 간 관계 네트워크
- 시간적 선후관계 (A → B)
- 의학적 인과관계 (A 때문에 B)
- 치료적 연관성 (A 치료로 B)
- 각 관계에 신뢰도 점수 부여
