## 목표
- 태그 사전(입원/수술/영상/검사) 강화로 태그 적중률 향상
- 골든 비교 자동화로 회귀 감시 정확성 향상
- 컨트롤러 opt-in 기본값을 기능 플래그로 통제

## 태그 사전 강화
- 사전 파일 추가: 동의어·영문/약어 매핑
- 추출 함수 도입: 텍스트→표준 태그 세트 반환
- 바인딩 단계에서 사전 기반 태그 검출로 교체

## 골든 비교 자동화
- outputs/ten-report/golden 아래 report.json과 현재 report.json 비교
- 결과 집계: 케이스별 diff 라인 수·불일치 여부 요약
- 산출: outputs/ten-report/_golden_diff.json/_golden_diff.md

## 기능 플래그 연계
- 컨트롤러에서 tenItemReport 기본값을 기능 플래그로 결정(쿼리 미지정 시)
- 플래그는 env로 오버라이드 가능

## 실행 순서
1) 태그 사전 파일/검출 함수 추가→dateBindingV3에 적용
2) 골든 비교 스크립트 추가→실행·요약 생성
3) 컨트롤러 플래그 연계 적용

## 산출물
- tagSynonyms 파일·검출 함수, golden 비교 스크립트, diff 요약 파일

## 위험·대응
- 태그 과적용: 사전 정규화 우선·룰 폴백 최소화
- 골든 불일치: 지표로 원인 확인 후 단계적 튜닝
